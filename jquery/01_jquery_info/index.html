jQuery 란?

제이쿼리는 자바스크립트의 생산성을 향상시켜주는 자바스크립트 라이브러리입니다. 자바스크립트 라이브러리란, 자주사용하는 코드들을 재사용 할 수 있는 형태로 가공해서 프로그래밍기법

jQuery 특징?

  1. 엘리먼트를 선택하는 강력한 기능 제공
  2. 선택된 엘리먼트들을 제어할 수 있는 다양한 함수
  3. 제공 자바스크립트를 라이브러리화 함.

jQuery 문법?

  1. $("대상").함수();
  2. jQuery("대상").함수();

jQuery 랩퍼(Wrapper)

  1. $(document).ready(function(){});
  2. $(function(){});

jQuery 문서 연결방법

  1. jQuery 직접연결시 : 제이쿼리 공식사이트에서 완성된버전의 최소버전을 다운받아서 연결한다.
  2. CDN 주소로 연결시 : 서비스를 제공하는 업체로부터 링크주소를 사용해서 문서에 연결하여 서비스 함.
